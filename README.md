# luasnips

This lua sources should be sources by the [LuaSnip](https://github.com/L3MON4D3/LuaSnip) `lazy_load` function as follows:

```lua
require("luasnip.loaders.from_lua").lazy_load { ["paths"] = "path/to/this/repo" }
```
